Перед установкой модулей желательно папке Lazarus и всему, что в ней находится дать полные
права или чтобы владелец папки Lazarus и всего, что внутри был тот кто будет пользоваться
Лазарусом, если этого не сделать, то при пересборке Лазарус соберётся в папку того
пользователя, который его пересобирает в итоге получится много путаницы с тем какой именно
Лазарус запускается, откуда он берёт параметры для компиляции, откуда он будет брать уже
скомпилиные модули, короче будет много проблем.
Тоже самое желательно сделать с файлом, который хранит параметры компиляции:
/etc/fpc.cfg
Папку “modeles” можно сохранить в любое месте, например
/home/projects/Lazarus/
После этого в файле fpc.cfg нужно прописать пути к модулям, чтобы компилятор знал, где искать
модули. Ищем строку в файле fpc.cfg – # searchpath for units and other system dependent things и
ниже дописываем пути к своим модулям:
-Fu/home/projects/Lazarus/modules
-Fu/home/projects/Lazarus/modules/linux
Лучше сделать резервную копию исполняемого файла Lazarus (он вроде самый большой), если
что-то пойдёт не так, то вернув исполняемый файл всё восстановится.
